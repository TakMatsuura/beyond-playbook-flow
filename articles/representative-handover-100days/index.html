代表交代直後の100日でやるべき、事業構造の棚卸し

代表交代直後の社長の頭の中は、カオスです。
借入・組織・顧客・契約・キャッシュ ── すべてが一気に押し寄せ、 ★何から手をつければいいか分からない★ という状態は珍しくありません。
本記事では、最初の100日で事業構造を5つの視点で棚卸しする方法を、BEYOND Holdingsの伴走思想とともに解説します。

1. なぜ「最初の100日」が勝負なのか

代表交代直後の100日は、新代表が組織と外部に対して「方針を見せる」期限です。100日を超えると、社員・取引先・銀行は「あの新社長は何を考えているか分からない」 と疑心を持ち始め、信用を失いやすくなります。

一方で、 ★この100日でいきなり全部を変えようとすると失敗します★。前代表の遺産を理解せず、感覚で動くと組織が反発し、取引先が離れます。

正しいアプローチは、 ★最初の100日は「棚卸し」 に徹し、その上で101日目以降の打ち手を発表する★ ことです。BEYONDが伴走する場合、この100日を5つの構造視点でガイドします。

2. 100日で棚卸しすべき、5つの構造視点

5つの構造視点
  1. お金の構造: BS/PL/CF + 借入残・銀行リレーション・資金繰り表
  2. 顧客の構造: 上位顧客集中度・売掛回収サイト・契約更新時期
  3. 組織の構造: キーパーソン・離職リスク・人件費構造・後継候補
  4. 商品/事業の構造: 商品別粗利・成長カテゴリ・撤退候補
  5. 契約・許認可の構造: 個人保証・取引契約・許認可・地代家賃契約

視点① お金の構造

BS/PL/CF の整合性確認 + 借入一覧・銀行担当者の関係性・資金繰り表の現状を整理。 ★試算表が遅れていれば数字作りから★。 ここを最優先で固めないと、他の判断ができません。

視点② 顧客の構造

上位3社で売上の何%か、契約更新が代表交代と重なっていないか、売掛金の回収サイトに無理がないか。 ★代表交代を理由に「契約見直し」を持ちかけてくる取引先★ もいるので、リスクを早期に把握。

視点③ 組織の構造

キーパーソンは誰か(離職するとどの業務が止まるか)、人件費が売上に占める比率、後継候補の有無。 ★代表交代直後3ヶ月以内の離職★ は典型的なリスク。事前に対話を仕掛ける。

視点④ 商品/事業の構造

商品別の粗利率・売上トレンド・原価構造を見て、「成長させるべき柱」 と「撤退検討対象」 を仕分け。前代表が大事にしていた事業ほど、客観的に見ると赤字垂れ流しのケースがあります。

視点⑤ 契約・許認可の構造

個人保証(代表交代で引き継ぐもの)、長期契約、許認可更新時期、地代家賃契約の見直し可能性。 ★個人保証の切替交渉★ は、新代表になってすぐ動くべき項目。

BEYONDが伴走する場合、こんなタイムラインで進めます。

★お金の構造 完全把握★
BS/PL/CF整理・借入一覧・銀行訪問・資金繰り表完成。 ★社長が毎朝10分で見れるダッシュボード★ をBEYONDが構築 (現預金・借入・売掛・在庫の週次変動)。
60日目
★顧客・組織の構造 把握 + キーパーソン対話★
上位顧客との面談・キーパーソンとの1on1・離職リスク特定・キャッシュ流入の予測精度UP。
100日目
★立て直しの軸 発表★
商品/事業・契約・許認可を含めた全体構造の棚卸し完了。 ★101日目以降の打ち手★ (撤退判断・新規投資・組織再編・銀行交渉) を、社内外に発表できる状態に。

4. BEYONDが伴走するなら、こう動く (想定事例)

SIMULATION

製造業 / 年商1億 / 創業者→二代目交代 + 債務超過状態

創業者から二代目への代表交代時。前代表時の借入数千万・債務超過・毎月のキャッシュは持って2ヶ月。新代表は「何から手をつければいいか分からない」 状態。

0-30日: BS/PL/CF整理。試算表が3ヶ月遅れていたので、数字作りから着手。 ★社長が毎朝10分で読めるダッシュボード★ を BEYOND が構築。借入一覧・銀行関係者リスト・資金繰り3ヶ月先予測まで完成。
30-60日: 上位3社との面談を BEYOND同席で実施。 ★代表交代を理由に発注を絞る動き★ が1社見られたため、新代表の方針を直接伝えて防止。社内のキーパーソン3名と1on1し、 ★うち1名が転職検討中★ と判明 → 早期に役割と待遇を見直して引き留めに成功。
60-100日: 商品別粗利を整理し、 ★赤字事業ライン1つの撤退判断★。同時に、 ★成長カテゴリへの集中投資★ を社内発表。個人保証は新代表名義への切替を銀行と交渉開始。
結果イメージ: 100日目に、 ★社内外に「新代表の方針」 を明確に発表できる状態★。借入の状況と立て直しシナリオを銀行に提示し、 ★リスケ合意 + 補助金採択★。月次キャッシュフロー黒字化までの道筋が見える。
